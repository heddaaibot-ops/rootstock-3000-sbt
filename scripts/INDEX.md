## ⚡ 3分钟快速了解v2.0

### 6大改进

1. **失败重试** - 自动重试3次，成功率↑
2. **断点续传** - 中断后继续，节省时间
3. **余额预检** - 提前检查，避免浪费
4. **批量检查** - 速度提升85%
5. **成本统计** - 详细报表，100%透明
6. **Token ID** - 自动提取保存

### 性能提升

- SBT状态检查: **26秒 → 3.8秒** (提速85%)
- 失败处理: **手动 → 自动重试**
- 中断恢复: **从头开始 → 断点续传**

### 使用方法（不变）

```bash
# 完全兼容v1.0，无需改变
PRIVATE_KEY=0x... node auto-mint.js all
```
