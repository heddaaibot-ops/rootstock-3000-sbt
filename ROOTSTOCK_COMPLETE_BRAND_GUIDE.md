## 🔤 字体系统（Typography Guidelines）

### 主字体：Rootstock Sans

**字符集**：
- 大写字母：`ABCDEFGHIJKLMNOPQRSTUVWXYZ`
- 小写字母：`abcdefghijklmnopqrstuvwxyz`
- 数字：`1234567890`
- 符号：`@!*#$%^&*()_+-=[]{}|;:',.<>?/`

### 字体使用规范

#### **标题层级**

**H1 - 大标题**：
- 字体：Rootstock Sans Bold
- 颜色：Lime / Orange / Pink（高对比色）
- 用途：页面主标题、英雄区标题

**H2 - 次标题**：
- 字体：Rootstock Sans Bold/Semi-bold
- 颜色：White / Orange
- 用途：段落标题

**H3 - 三级标题**：
- 字体：Rootstock Sans Regular/Semi-bold
- 颜色：根据背景选择
- 用途：卡片标题、模块标题

#### **正文**

**Body Text**：
- 字体：Rootstock Sans Regular
- 颜色：White (黑底) / Black (白底)
- 行高：1.5-1.6
- 字间距：正常

**Caption / Small Text**：
- 字体：Rootstock Sans Regular
- 字号：小
- 用途：说明文字、标签文字

### Letter Modes（字母模式）

从指南可以看到两种文字处理方式：

1. **Outlined Mode（描边模式）**
   - 在深色背景使用
   - 黑底 + 荧光色文字

2. **Published Mode（发布模式）**
   - 实际应用的文字
   - 带有颜色高亮（黄色/橙色背景 + 黑色文字）

### Typography In Context（实际应用）

**示例场景**：
- "Build for billions" - 橙色底 + 黑色粗体
- "Think like us?" - 黑底 + 多色文字
- "Meet the dApps built on Bitcoin" - 绿色高亮 + 黑色文字

**规则**：
- 标题文字使用高亮色块背景（Lime/Orange）
- 正文保持黑白对比
- 强调词使用彩色下划线或背景

### Leading and Tracking Space（行距和字距）

从 Typography 页面右侧图示可见：
- **Leading（行距）**：充足的行间距
- **Tracking（字距）**：标准或略宽

## 🎨 Logo 系统（Logo Guidelines）

### Logo 格式

#### **可用格式**

1. **SVG - Lock-up**（文字 + 图标组合）
   - 橙色版本：`Rootstock Sans` 文字 + 花形图标
   - 黑色版本：全黑色

2. **SVG - Logo**（仅图标）
   - 橙色花形图标
   - 黑色花形图标

3. **PNG - Lock-up**
   - 橙色版本
   - 黑色版本

4. **PNG - Logo**
   - 橙色花形图标
   - 黑色花形图标

### Logo 构成

**花形图标特征**：
- 8个圆形组成的花形/树根形状
- 主色：Rootstock Orange (`#FF9100`)
- 可单独使用或与文字组合

**文字部分**：
- 字体：Rootstock Sans
- "Rootstock" - 全小写或首字母大写
- 颜色：Orange / Black

### Logo 使用场景

从指南中可见多种应用：

1. **硬件设备**
   - 钱包硬件上的图标
   - 黑白或橙色版本

2. **移动应用**
   - App 图标
   - 橙色主视觉

3. **印刷品**
   - 海报、传单
   - T恤、帆布袋
   - 笔记本、贴纸

4. **数字媒体**
   - 社交媒体头像
   - 网站 favicon
   - 展示屏幕

### Logo 规范

- ✅ 保持清晰可辨
- ✅ 预留足够空白空间
- ✅ 在浅色背景用黑色版本
- ✅ 在深色背景用橙色版本
- ❌ 不要拉伸变形
- ❌ 不要改变配色
- ❌ 不要旋转使用

## 🎨 插画系统（Illustration Guidelines）

### 插画风格

从 Illustration 页面可以看到完整的插画样式系统：

#### **样式特征**

1. **几何抽象风格**
   - 使用简单几何形状
   - 网格和线条组合
   - 等距视角（Isometric）

2. **配色方案**
   - 黑色背景为主
   - 橙色作为主视觉元素
   - 辅以紫色、绿色、青色

3. **元素类型**
   - 建筑/结构图形
   - 数据可视化图表
   - 抽象图案装饰

### 插画应用场景

从指南可见多种应用：

**1. 信息图表（Infographics）**
- 柱状图、折线图
- 橙色数据条
- 黑色或白色背景

**2. 装饰性插画（Decorative）**
- 网格图案
- 抽象几何形状
- 建筑透视图

**3. 概念插画（Conceptual）**
- 立体建筑物
- 连接节点网络
- 数据流动示意

### 插画使用规则

- ✅ 使用官方配色系统
- ✅ 保持几何简洁风格
- ✅ 与文字内容呼应
- ✅ 适当留白
- ❌ 不要使用渐变（除非官方插画包含）
- ❌ 不要使用照片拼贴
- ❌ 不要偏离品牌色系

## 🏷️ Nametags 系统（Nametags Guidelines）

### Nametag 概念

**Nametags** = 用户名标签系统，类似 Web3 用户名牌

从指南可以看到：

### 设计规范

#### **基础样式**

**荧光黄绿色背景**（Lime）：
```css
background: #DEFF1A;
color: #000000;
border-radius: 60px; /* 胶囊形状 */
padding: 10px 30px;
font: Rootstock Sans Bold;
text-transform: uppercase; /* 全大写 */
```

**示例**：
- `NIRK333` - 黄绿色胶囊形标签
- 黑色粗体文字
- 居中对齐

#### **变体样式**

从指南可见多种配色方案：

1. **主样式 - Lime**
   - 背景：`#DEFF1A`（荧光黄绿）
   - 文字：Black

2. **次要样式 - Orange**
   - 背景：`#FF9100`
   - 文字：Black

3. **深色样式 - Green**
   - 背景：`#78C700`（深绿）
   - 文字：Black

4. **其他配色**
   - Purple 背景
   - Pink 背景
   - Cyan 背景

### Nametag 应用场景

从 Brand In Use 页面可以看到实际应用：

1. **社交媒体用户名**
2. **Discord/Telegram 身份标识**
3. **活动名牌/胸牌**
4. **Web3 钱包地址显示**
5. **NFT 持有者标识**

### 使用规范

- ✅ 用户名全大写
- ✅ 使用胶囊形圆角
- ✅ 高对比度配色（亮色背景 + 黑字）
- ✅ 粗体字重
- ❌ 不要使用细字体
- ❌ 不要用小写字母
- ❌ 不要使用低对比度配色

## 🌍 品牌应用实例（Brand In Use）

### 实际应用场景

从 Brand In Use 页面可以看到 Rootstock 的实际应用案例：

#### **1. 数字媒体**

**社交媒体帖子**：
- 黑色背景
- 荧光色标题（Lime/Orange）
- 多色标签组合
- 示例："Build for billions" - 橙色高亮

**网站界面**：
- 深色主题
- 橙色 CTA 按钮
- 绿色荧光标题
- 示例："Meet the dApps built on Bitcoin"

**移动应用**：
- 黑底界面
- 彩色卡片
- 橙色花形 Logo

#### **2. 印刷品**

**海报/广告牌**：
- "Harder Better Faster Stronger" - 多色条纹背景
- "Opening up the world with open source" - 粉色主视觉
- 黑底 + 荧光色大字

**户外广告**：
- 大型展示屏
- 黑底橙字
- 高对比度确保可读性

#### **3. 实物周边**

**T恤/帆布袋**：
- 白色或黑色底
- 橙色花形 Logo
- 或多色印刷图案

**硬件设备**：
- 钱包设备上的 Logo
- 简洁单色或橙色

**文具用品**：
- 笔记本封面
- 贴纸
- 名片

### 案例学习

#### **案例 1：Taringai 社交网络**

从指南中的案例：
> "Taringai is using Rootstock to create Latin America's largest social network"

设计特征：
- 黑色背景
- 黄色文字高亮
- 立体建筑插画（黑白风格）
- Rootstock Logo 点缀

#### **案例 2：Smart Contract 推广**

设计特征：
- 荧光绿色背景卡片
- 黑色文字
- 多色装饰元素
- 简洁信息呈现

### 品牌应用原则

1. **保持一致性**
   - 使用官方配色
   - 遵循字体规范
   - 统一视觉语言

2. **高对比度**
   - 确保可读性
   - 黑白对比为主
   - 彩色强调为辅

3. **大胆使用色彩**
   - 不怕使用荧光色
   - 多色组合营造活力
   - 保持品牌识别度

4. **简洁有力**
   - 信息清晰直接
   - 视觉冲击力强
   - 避免过度装饰

## 🎯 为 Rootstock 3000 SBT 应用指南

### 推荐设计方案

基于完整品牌指南分析，我们的 SBT DApp 应该这样设计：

#### **配色方案**

```css
/* 主题配色 */
:root {
  /* 背景色 */
  --bg-primary: #000000;      /* Black */
  --bg-secondary: #1a1a1a;    /* 深灰 */
  --bg-tertiary: #2a2a2a;     /* 中灰 */

  /* 文字色 */
  --text-primary: #FAFAF5;    /* Off White */
  --text-secondary: #a0a0a0;  /* 浅灰 */
  --text-inverse: #000000;    /* Black */

  /* 强调色 */
  --accent-primary: #FF9100;  /* Rootstock Orange - 主按钮 */
  --accent-secondary: #FF70E0; /* Rootstock Pink - 次按钮 */
  --accent-title: #DEFF1A;    /* Lime - 标题 */
  --accent-success: #78C700;  /* Green - 成功状态 */

  /* 辅助色 */
  --purple: #9E75FF;
  --cyan: #08FFD1;
  --lime: #DEFF1A;

  /* 圆角 */
  --radius-sm: 4px;
  --radius-md: 13.65px;
  --radius-lg: 26.20px;

  /* 阴影 */
  --shadow-sm: 0 2px 8px rgba(255, 145, 0, 0.1);
  --shadow-md: 0 4px 16px rgba(255, 145, 0, 0.2);
  --shadow-lg: 0 8px 32px rgba(255, 145, 0, 0.3);
}
```

#### **组件设计**

**1. 主按钮（Mint Button）**
```css
.btn-primary {
  background: #FF9100;  /* Rootstock Orange */
  color: #000000;
  border-radius: 13.65px;
  padding: 16px 32px;
  font-family: 'Rootstock Sans', sans-serif;
  font-weight: bold;
  text-transform: uppercase;
  box-shadow: 0 4px 16px rgba(255, 145, 0, 0.4);
}

.btn-primary:hover {
  background: #ff9d1a;
  box-shadow: 0 6px 20px rgba(255, 145, 0, 0.6);
  transform: translateY(-2px);
}
```

**2. 卡片（Stats Card）**
```css
.card {
  background: #000000;
  border: 2px solid #FF9100;
  border-radius: 4px;
  padding: 24px;
  box-shadow: 0 4px 16px rgba(255, 145, 0, 0.2);
}
```

**3. 标题（Page Title）**
```css
.title {
  font-family: 'Rootstock Sans', sans-serif;
  font-size: 48px;
  font-weight: bold;
  color: #DEFF1A;  /* Lime */
  text-transform: uppercase;
  letter-spacing: -0.02em;
}
```

**4. 进度条（Mint Progress）**
```css
.progress {
  background: #1a1a1a;
  height: 8px;
  border-radius: 4px;
  overflow: hidden;
}

.progress-fill {
  background: linear-gradient(90deg, #FF9100, #DEFF1A);
  height: 100%;
  transition: width 0.3s ease;
}
```

**5. 标签（Tags/Badges）**
```css
.badge {
  background: #FF70E0;  /* Pink */
  color: #000000;
  border-radius: 13.65px;
  padding: 4px 12px;
  font-size: 12px;
  font-weight: bold;
  text-transform: uppercase;
}
```

**6. 用户名标签（Nametag）**
```css
.nametag {
  background: #DEFF1A;  /* Lime */
  color: #000000;
  border-radius: 60px;
  padding: 10px 24px;
  font-family: 'Rootstock Sans', sans-serif;
  font-weight: bold;
  text-transform: uppercase;
  display: inline-block;
}
```

#### **页面布局建议**

**Hero Section**：
- 黑色背景
- 荧光黄绿色大标题："ROOTSTOCK 3000 DAYS"
- 橙色副标题

- 背景可加入几何插画元素

**Stats Section**：
- 黑色卡片 + 橙色边框

**Footer**：
